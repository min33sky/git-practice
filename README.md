## 브랜치

1. Remote Branch에 Push 하기

```bash
git push -u origin(ex.원격 저장소) [branch-name]
```

2. All Branch List

```bash
git branch -a
```

3. Remote Branch Delete

```bash
git push origin(ex.원격 저장소) --delete [remote branch-name]
```
